# 😎 Donghoon Shin - Personal website

My personal website theme, based upon [Autumn](https://github.com/connor-baer/autumn) theme!

## Usage

You'll need to have [Jekyll](https://jekyllrb.com/), the `bundler` gem, and [`yarn`](https://yarnpkg.com/) (`npm` also works) installed on the computer.

```bash
bundle install
bundle exec jekyll serve
```

## Options

Donghoon theme includes some customizable options, applied via options in the `_config.yml` file. A standout feature is the dark theme.
Have a look at the [`_config.yml`](_config.yml) file for all the available options.
